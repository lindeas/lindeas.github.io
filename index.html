JILO - Jitsi Logs Observer

Bash scripts for collecting and displaying information about conference events from Jitsi Meet logs.

Includes:

  • jilo - server side tool for gathering data from Jitsi logs
  • jilo-cli - command line client for querying the database of collected logs data

Jilo Web - Jitsi logs web observer

Web interface to a Jilo database.

The webpage for this project is https://lindeas.com/jilo. There you can find the git code mirrors and other info.
